HAL名古屋に公式サイトをリメイクしました。
コンセプト/デザインの方向性は、無限の可能性を最大限引き出し、一人一人輝ける未来へ。そして、このコンセプトにそってサイト全体のテーマを宇宙にしてみました。拘った部分は、スマホのレスポンシブです。例えば、３つの画像を横並びして表示させてある広告バナーを、スマホサイズでは３つの画像を順番に表示させるスライド式にして見やすくさせています。他にもbackground-imageを固定にして立体のように見せたり、header-navをスマホサイズ時はハンバーガーメニューで表したりしています。サイトをご覧になるときはぜひレスポンシブサイトにも注目して見てください。

完成日:2021年2月下旬
言語：HTML, CSS3, javascript
使用ソフト：Photoshop xd
